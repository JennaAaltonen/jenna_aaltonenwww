A couple of differences between HTML and HTML5:

  1. With HTML5 you can use new elements including video, time, nav, section, progress, meter, aside and canvas
  2. With HTML5 the interactive media, such as games, can run in web browsers without the need for extra software or plugins. Audio and video playback also no longer require additional plugins.
  3. With HTML5 some data can be stored on the user’s device, which means apps can continue working properly without an Internet connection.
